# poc-api

| Lang| Version |
| ----------- | ----------- |
| Python      | 3.10.4       |

## Starting environment
Windows users

```
$ py -m pip install --user virtualenv 
$ py -m venv env   
$ .\env\Scripts\activate  
```

Unix
```
$ python3 -m venv env
$ source env/bin/activate
```

## Installing dependencies & running
```
$ pip install -r requirements.txt
$ python3 main.py
$ sudo chmod +x main.py 		# Optional
$ ./main.py 					# Optional

```
